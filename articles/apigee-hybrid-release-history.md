# バージョニング

`Major.Minor.Patch`

マイナーは四半期ごと、パッチは毎月提供されます。
[Apigee リリース プロセス](https://docs.apigee.com/release/apigee-edge-release-process?hl=ja#apigee-hybrid-release-process) に記載があります。
